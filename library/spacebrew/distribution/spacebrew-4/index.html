spacebrewP5

A Library by Brett Renfer for the Processing programming environment.
Last update, 10/15/2015.

Spacebrew is a toolkit for prototyping interactive spaces.
SpacebrewP5 allows you to connect your Sketches to Spacebrew. It wraps simple WebSocket client functionality, and allows for sending + receiving of messages through Spacebrew.

Download

Download spacebrewP5 version 0.4.1 (4) in .zip format.

Installation

Unzip and put the extracted spacebrew folder into the libraries folder of your Processing sketches. Reference and examples are included in the spacebrew folder.
